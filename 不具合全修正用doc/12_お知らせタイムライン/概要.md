# お知らせの全コース横断タイムライン / 概要

## 機能および要件

- ヘッダーにお知らせボタンを追加し、クリックでモーダルを開けること
- 全コースの Announcement API を並列に叩き、`createdOn` で時系列ソートすること
- 各お知らせ: コース名・タイトル・日時・本文プレビュー（折りたたみ展開）を表示すること
- 新着（前回表示以降）にはバッジを付けること
- コース名でのフィルタ機能があること
- 開いたお知らせは既読にすること

## 取得元・Fetch要件

- Announcement API: `GET {baseURL}/direct/announcement/site/{courseId}.json?n=100`（全コース並列）
- 使用フィールド: `title`, `body`, `createdOn`, `announcementId`, `siteId`

## 保存要件

- `chrome.storage.local[hostname]["announcementLastSeen"]`: 最終閲覧時刻
- `chrome.storage.local[hostname]["readAnnouncements"]`: 既読お知らせ ID リスト

## UI 配置

- 時間割ボタン・カレンダー同期ボタンと同じ Sakai ヘッダー領域にボタンを配置
- モーダル表示は既存モーダルと同じ方式

## 未実装要件（将来検討）

- 重要マーク / ピン留め機能
- お知らせの既読一括管理
