# TACTメモタブ / 検索タブ

## 機能および要件

- キーワード入力でメモ検索できること
- 検索ボタンまたは Enter で検索を実行できること
- 2文字未満では検索を抑制できること
- 検索結果一覧から対象メモを確認できること

## 取得元・Fetch要件

- `localStorage["tact-lecture-notes"]` の全メモを検索対象とすること
- 検索対象フィールドは `note`, `lectureName`, `links[].title`, `links[].url`, `links[].description` であること
- ネットワーク `fetch` は不要であること

## 保存要件

- 検索タブ専用の保存キーは持たず、`tact-lecture-notes` を都度検索すること
- 検索履歴の保存は現状行わないこと

- リンク URL 検索
- 更新日や講義別の絞り込み
- 検索履歴

## 不具合（後ほど調査）
