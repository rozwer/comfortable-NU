# TACTメモタブ / 現在の講義タブ

## 機能および要件

- 現在の講義に紐づくメモ一覧を表示できること
- メモが 0 件のとき、空状態を表示できること
- 空状態から新規メモ作成へ遷移できること
- メモ項目をクリックして関連リンクを開けること

## 取得元・Fetch要件

- `localStorage["tact-lecture-notes"]` から全メモを読み、その中から `lectureId === currentLectureId` のものだけ表示すること
- `currentLectureId` は URL `/site/{lectureId}` から抽出すること
- 外部ネットワーク `fetch` は不要であること

## 保存要件

- 新規作成、編集、削除はすべて `localStorage["tact-lecture-notes"]` を更新すること
- 現在の講義タブ専用の保存キーは持たず、講義 ID フィルタで再構成すること

## 未実装要件

- 現在講義内メモの並び替え
- 現在講義内メモの絞り込み
- 重要メモの固定表示

## 不具合（後ほど調査）

- 現在講義判定が TACT の URL や DOM 差分でずれないか確認
- 単一リンクと複数リンクのクリック挙動が一貫しているか確認
