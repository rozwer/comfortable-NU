# 横断ダッシュボード（週次ビュー） / 概要

## 機能および要件

- ヘッダーにダッシュボードボタンを追加し、クリックでモーダルを開けること
- 全コースの課題・小テストを週単位のガントチャート風に表示すること
- 横軸: 月〜日（1 週間）、縦軸: 課題名
- `openTime` 〜 `dueTime` の期間を横バーで表示すること
- 色分け: 未提出（赤系）/ 提出済み（緑系）/ 小テスト（青系）
- 週の切替（前週・次週）ができること
- 課題名クリックで該当コースページへ遷移すること

## 取得元・Fetch要件

- Assignment API: `GET {baseURL}/direct/assignment/site/{courseId}.json`（全コース並列）
- Quiz API: `GET {baseURL}/direct/sam_pub/context/{courseId}.json`（全コース並列）
- 使用フィールド: `dueTime`, `openTime`, `closeTime`, `title`, `submissions`（Assignment）、`dueDate`, `title`（Quiz）

## 保存要件

- 表示データは毎回 API から取得するためキャッシュ不要（既存の miniSakai キャッシュを流用可）

## UI 配置

- 時間割ボタン・カレンダー同期ボタンと同じ Sakai ヘッダー領域に配置
- モーダル表示は既存の時間割モーダルと同じ方式
