# 提出率トラッカー / 概要

## 機能および要件

- ヘッダーに提出率ボタンを追加し、クリックでモーダルを開けること
- コースごとの提出率を横棒グラフで表示すること（提出済み / 未提出 / 合計）
- 全体の提出率サマリーも表示すること
- 未提出課題の一覧を展開表示可能であること

## 取得元・Fetch要件

- Assignment API: `GET {baseURL}/direct/assignment/site/{courseId}.json`（全コース並列）
- 使用フィールド: `submissions`, `title`, `dueTime`

## 保存要件

- 表示データは毎回 API から取得するためキャッシュ不要

## UI 配置

- 時間割ボタン・カレンダー同期ボタンと同じ Sakai ヘッダー領域にボタンを配置
- モーダル表示は既存モーダルと同じ方式

## 未実装要件（将来検討）

- 提出率の推移グラフ（時系列）
- 成績連動（Grade API がある場合）
