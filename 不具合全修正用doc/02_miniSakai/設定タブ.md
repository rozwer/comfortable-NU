# miniSakai / 設定タブ

## 機能および要件

- ダークテーマを切り替えられること
- 遅延提出課題の表示可否を切り替えられること
- 未公開課題の表示可否を切り替えられること
- 課題キャッシュ間隔を変更できること
- クイズキャッシュ間隔を変更できること
- 色設定を変更できること
  - 上部タブ色
  - miniSakai色
  - 時間割色
- 色設定を初期値へ戻せること
- 設定変更後に miniSakai と時間割モーダルへ反映されること

## 取得元・Fetch要件

- 設定読込は `getStoredSettings(hostname)` 経由で行うこと
- 設定読込時に `chrome.storage.local[hostname]["AssignmentFetchTime"]` と `["QuizFetchTime"]` を併読し、UI に取得時刻を反映できること
- 外部ネットワーク `fetch` は不要で、設定表示は保存済みオブジェクトから復元すること

## 保存要件

- `chrome.storage.local[hostname]["Settings"]`: 設定オブジェクト全体の保存先
- 保存単位は差分キーではなく `Settings` オブジェクト丸ごとであること
- 色設定リセット時も同じ `Settings` キーへ再保存すること
- 設定保存後に miniSakai と時間割モーダルへ CSS カスタムプロパティを再適用すること

## 未実装要件
