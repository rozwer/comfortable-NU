# miniSakai / 非表示タブ

## 機能および要件

- 将来日時付きで非表示化されたエントリを一覧表示できること
- コースリンクから対象講義へ移動できること
- 締切日時、残り時間、`非表示` バッジを表示できること
- `×` ボタンで非表示状態を解除し、通常一覧へ戻せること

## 取得元・Fetch要件

- このタブは追加のネットワーク `fetch` を行わず、miniSakai が保持しているエントリ配列を再利用すること
- 表示対象は `entry.checkTimestamp` を `yyyy/mm/dd/hh` として解釈し、現在時刻より未来であるエントリに限定すること
- コース情報は元の entity 配列から逆引きして付与すること

## 保存要件

- 読み出し元は `chrome.storage.local[hostname]["Assignments"]`, `["Quizzes"]`, `["Memos"]`
- `×` ボタン押下時は `entry.save(hostname)` を通して元の保存先の `checkTimestamp` を `undefined` に戻すこと
- 非表示タブ専用の保存キーは持たず、元データの `checkTimestamp` から毎回再計算すること
