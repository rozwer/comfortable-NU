# TACTフォルダタブ / 授業資料タブ

## 機能および要件

- フォルダ構造を取得して表示できること
- 再読み込みできること
- フォルダを開閉できること
- ファイルを選択できること
- 選択ファイルを一括ダウンロードできること
- 個別ファイルリンクを開けること
- ファイル名編集ができること
- NUSS ファイルは別処理で取得または案内できること

## 取得元・Fetch要件

- 資料一覧の取得元は `GET https://tact.ac.thers.ac.jp/direct/content/site/{siteId}.json`
- 資料ツリー生成時は `content_collection` を `FileSystemNode` 化すること
- 通常ファイルダウンロードは `fetch(file.url, { credentials: 'include' })` を使うこと
- NUSS ファイルは background へ `FETCH_NUSS_FILE` を送り、`downloadUrl = webLinkUrl + '/download'` で取得を試みること

## 保存要件

- `localStorage["tact-file-system-${siteId}"]`: ノード本体、表示名変更、移動、表示状態などの保存先
- `localStorage["folder-structure-cache-${siteId}"]`: 生成済みツリーのキャッシュ
- `localStorage["folder-structure-cache-time-${siteId}"]`: キャッシュ生成時刻
- ノード単位で `displayName`, `displayContainer`, `visible`, `localChanges` を保持すること
