# TACTフォルダタブ / 課題タブ

## 機能および要件

- 課題一覧を取得して表示できること
- 再読み込みできること
- 名前順、提出期限順、公開日順でソートできること
- `未提出を上部に表示` を切り替えられること
- 課題カードをクリックして詳細を開けること
- 詳細内で提出状態、期限、遅延提出、再提出を確認できること
- 添付ファイルを選択してダウンロードできること
- 詳細を閉じられること

## 取得元・Fetch要件

- 一覧取得元は `GET {baseURL}/direct/assignment/site/{courseId}.json?n=100`
- 詳細取得元は `GET {baseURL}/direct/assignment/{assignmentId}.json`
- 添付ファイル取得は `fetch(attachment.url, { credentials: 'include' })`
- NUSS 添付は通常資料タブと同様に URL 判定して別処理すること

## 保存要件

- `localStorage["assignment-cache-${courseId}"]`: 課題一覧キャッシュ
- `localStorage["assignment-cache-time-${courseId}"]`: 一覧キャッシュ時刻
- `localStorage["assignment-detail-${assignmentId}"]`: 課題詳細キャッシュ
- `localStorage["assignment-detail-time-${assignmentId}"]`: 詳細キャッシュ時刻
- ソート条件と `未提出を上部に表示` 状態は現状永続化しないこと
