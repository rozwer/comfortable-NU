# TACTフォルダタブ / お知らせタブ

## 機能および要件

- お知らせ一覧を取得して表示できること
- 再読み込みできること
- お知らせカードをクリックして詳細を開けること
- 投稿日、投稿者、本文、添付を確認できること
- 詳細を閉じられること

## 取得元・Fetch要件

- 一覧取得元は `GET {baseURL}/direct/announcement/site/{courseId}.json?n=100`
- 詳細表示は追加 `fetch` を行わず、一覧取得済みの `announcement_collection` をメモリ保持して再利用すること
- 本文プレビューでは HTML タグ除去後の短縮本文を表示すること

## 保存要件

- `localStorage["announcement-cache-${courseId}"]`: お知らせ一覧キャッシュ
- `localStorage["announcement-cache-time-${courseId}"]`: お知らせ一覧キャッシュ時刻
- `this.announcements`: 一覧取得後のメモリ保持先であり、永続化キーではないこと
