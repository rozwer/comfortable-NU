# 全教科横断フォルダブラウザ / 概要

## 機能および要件

- ヘッダーにフォルダボタンを追加し、クリックでモーダルを開けること
- フォルダ構造: `/{学期+季節}/{教科名}/...`（例: `2025春/線形代数/第1回/資料.pdf`）
- ルートから全教科のファイルツリーを一望できること
- 学期・季節による自動分類（TACT のコース情報から学期を判定）ができること
- フォルダ開閉ができること
- ファイル個別リンクで該当ページへ遷移できること
- 選択一括ダウンロード（ZIP）ができること
- 新着ファイルには「NEW」バッジを表示すること（新着ファイル検知と連動）

### 検索機能

- ファイル名・フォルダ名でインクリメンタル検索ができること
- 検索結果はフラットリスト表示（フルパスを表示）すること
- 検索ヒット箇所をハイライトすること
- 検索は全学期・全教科を横断すること

## 取得元・Fetch要件

- Content API: `GET {baseURL}/direct/content/site/{siteId}.json`（全コース並列）
- Course API: 学期・季節情報の取得（コース名またはコースメタデータから判定）
- `content_collection` 全体を取得して `FileSystemNode` 化すること

## 保存要件

- `localStorage["cross-folder-cache"]`: 全教科統合ツリーのキャッシュ
- `localStorage["cross-folder-cache-time"]`: キャッシュ生成時刻
- 各教科のファイルツリーは既存の `localStorage["tact-file-system-${siteId}"]` を流用

## UI 配置

- 時間割ボタン・カレンダー同期ボタンと同じ Sakai ヘッダー領域にボタンを配置
- モーダル表示は既存モーダルと同じ方式
