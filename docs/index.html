Sakai LMSを快適に

課題やクイズの管理を効率化し、学習体験を向上させるブラウザ拡張機能

今すぐダウンロード GitHub

主な機能

🎨

カラーコーディング

課題の期限に応じてコースサイトのタブを色分け表示。一目で緊急度がわかります。

  • 期限1日以内
  • 期限5日以内
  • 期限14日以内
  • 期限14日超
🔔

通知バッジ

未確認の最新課題を通知バッジで表示。重要な課題を見逃しません。

📋

miniSakai

すべての課題とクイズを一覧表示。カスタムメモも追加でき、チェックボックスで進捗管理も可能。

⚡

キャッシュ機能

ネットワーク負荷を軽減するキャッシュ機能を実装。設定により間隔を調整可能。
