機能一覧

ポータル画面とコースページを拡張する 15 の機能

01

Sakai ヘッダー拡張 CORE

ヘッダーに各機能のボタンを追加。miniSakai・時間割・カレンダー同期・ダッシュボードなどをワンクリックで起動。

02

miniSakai CORE

全コースの課題・小テスト・メモをサイドパネルに一覧表示。

  • 課題一覧（締切色分け・残り時間）
  • 提出済み・非表示タブ
  • カスタムメモ追加
  • 設定タブ（配色・表示切替）
03

subSakai

拡張アイコンから開くポップアップ版 miniSakai。TACT を開かずに課題・メモを確認可能。読み取り専用。

04

コースバー拡張

コースタブを締切カテゴリで色分け。未読課題への通知バッジも表示。

1日以内 5日以内 14日以内 14日超
05

時間割モーダル

登録講義を時間割形式で表示。年度・学期切替、教科色カスタマイズ、教室名入力、PNG エクスポートに対応。履修取り消し連動。

06

カレンダー同期

課題・小テストの締切を Google カレンダーに同期。手動/自動同期、同期間隔設定、重複防止機能つき。

07

TACT メモタブ TACT

コースページにメモタブを追加。現在の講義・全講義・検索の 3 タブでメモと外部リンクを管理。

08

TACT フォルダタブ TACT

コースページに資料フォルダタブを追加。授業資料・課題・お知らせの 3 タブでファイルをツリー表示。一括ダウンロード対応。

09

横断ダッシュボード V2

全コースの課題・小テストを週単位のガントチャート風に表示。未提出/提出済み/小テストを色分け。

10

新着ファイル検知 V2

講義資料の更新を自動検知し、該当コースにバッジ表示。background アラームで定期チェック。

11

お知らせタイムライン V2

全コースのお知らせを時系列で一覧表示。新着バッジ、本文プレビュー展開、コースフィルタ。

12

学習負荷可視化 V2

今後 4 週間の締切数をカレンダー型ヒートマップで表示。日ごとの詳細をホバーで確認可能。

13

提出率トラッカー V2

コースごとの提出率を横棒グラフで可視化。全体サマリーと未提出課題の展開表示に対応。

14

横断フォルダブラウザ V2

全コースのファイルを学期/教科で自動分類。インクリメンタル検索、ZIP 一括ダウンロード、新着バッジ。

15

履修取り消し

不要なコースを非表示にし、時間割・プレビューから除外。各機能と連動して一貫した非表示を実現。

Specification

機能詳細

各機能の仕様と対応範囲

01

Sakai ヘッダー拡張

Sakai ポータルのヘッダー領域に miniSakai・時間割・カレンダー同期・ダッシュボード・お知らせ・負荷・提出率・フォルダの各ボタンを挿入します。ログイン済み画面でのみ表示され、DOM への直接挿入で動作します。

02

miniSakai

Sakai 画面上にサイドパネルとして表示される中核機能。全コースの課題・小テスト・メモを一元管理します。

課題一覧タブ

全コースの課題・クイズを締切順に表示。残り時間、再提出可否バッジ、コースページリンクを表示。緊急度に応じて色分け。

提出済みタブ

提出完了した課題を一覧表示。提出日時を確認可能。

非表示タブ

マイナスボタンで非表示にした課題の一覧。復元操作が可能。

設定タブ

配色テーマ、表示件数、通知設定を変更。即座に反映。

データ取得は Assignment API・Quiz API をコースごとに並列実行。一部コースが失敗しても残りで描画を継続します。

03

subSakai

拡張アイコンクリックで開くポップアップ UI。TACT ページを開かずに課題・小テスト・メモを確認できます。保存済みデータのみで動作する読み取り専用モードで、ネットワーク通信は発生しません。

04

コースバー拡張

Sakai のお気に入りコースバーを拡張。各コースタブを締切カテゴリ（1日以内=赤 / 5日以内=黄 / 14日以内=緑 / それ以上=灰）で色分けし、未読課題には通知バッジを表示します。miniSakai の課題データをもとに算出。

05

時間割モーダル

登録講義を時間割表形式で表示するモーダル。年度・学期の切替、お気に入りのみ/全コース切替に対応。

  • 教科色設定: 教科ごとの色変更・教室名入力・個別/全体リセット
  • PNG エクスポート: 出力サイズ選択付き
  • 履修取り消し連動: 非表示コースを時間割から自動除外

データはポータル DOM（a.link-container[title] 等）から解析。ネットワーク fetch は不使用。

06

カレンダー同期モーダル

Google カレンダーとの連携モーダル。課題・小テストの締切をカレンダーイベントとして自動登録。

  • Google アカウント認証（OAuth 2.0）
  • 課題同期・クイズ同期を個別に ON/OFF
  • 手動同期 / 自動同期（間隔設定可能）
  • 送信済みキーによる重複登録防止
  • 最終同期時刻の表示・送信履歴クリア
07

TACT メモタブ

TACT コースページのツールメニューにメモ機能を追加。講義ごとのメモ本文と関連リンクを保存。

現在の講義タブ

閲覧中のコースに関連するメモを表示・編集。

全講義タブ

全コースのメモを横断的に一覧表示。

検索タブ

メモ内容をキーワードで検索。

データは localStorage に保存。ネットワーク通信なしで動作。

08

TACT フォルダタブ

TACT コースページに資料フォルダ機能を追加。Content API / Assignment API / Announcement API からデータを取得し、ツリー形式で表示。

授業資料タブ

講義資料をフォルダ階層で表示。個別/一括ダウンロード対応。

課題タブ

課題一覧と詳細（添付ファイル含む）を表示。

お知らせタブ

コースのお知らせ一覧を表示。

キャッシュ TTL 2 時間。再読み込みボタンからの手動更新にも対応。

09

横断事項

各機能が共有するインフラストラクチャ。

  • ストレージ: chrome.storage.local（ホスト別構造）と localStorage（ページローカル）を併用
  • API: Sakai REST API（Assignment / Quiz / Content / Announcement）+ Google OAuth2 / Calendar API
  • DOM 解析: .fav-sites-entry / .link-container / #toolMenu ul からの情報抽出
  • 対応ブラウザ: Chrome（Manifest V3）。Firefox / Edge / Safari は将来対応
10

横断ダッシュボード

全コースの課題・小テストを週単位のガントチャート風ビューで表示。横軸が月〜日、縦軸が課題名で、openTime〜dueTime の期間を横バーで表現。未提出（赤系）/ 提出済み（緑系）/ 小テスト（青系）の色分け。週の前後切替が可能。

11

新着ファイル検知

Content API の modifiedTime / createdTime を前回取得分と比較し、新着ファイルを自動検出。該当コースのサイドバーにバッジを表示し、フォルダ閲覧で自動消去。background アラームによる定期チェック（間隔設定可能）。

12

お知らせタイムライン

全コースの Announcement API を並列取得し、createdOn で時系列ソートした横断タイムラインを表示。各お知らせにコース名・タイトル・日時・本文プレビュー（折りたたみ展開）。新着バッジと既読管理、コース名フィルタ機能つき。

13

学習負荷可視化

今後 4 週間分の締切数をカレンダー型ヒートマップで表示。日ごとの課題・小テスト件数をセルの色濃度で表現し、ホバーで詳細を確認可能。週ごとの合計件数も併記。

14

提出率トラッカー

コースごとの提出率を横棒グラフで可視化。全体の提出率サマリーを表示し、各コースの未提出課題一覧を展開表示可能。

15

横断フォルダブラウザ

全コースのファイルを /{学期+季節}/{教科名}/... の形式で自動分類したツリーで表示。TACT のコース情報から学期を自動判定。

  • ファイル名・フォルダ名でのインクリメンタル検索（全学期・全教科横断）
  • 検索結果はフルパス付きフラットリスト + ハイライト
  • 選択一括ダウンロード（ZIP）
  • 新着ファイル検知と連動した「NEW」バッジ表示
