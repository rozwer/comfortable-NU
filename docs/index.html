名古屋大学のSakai LMS（TACT）を快適に

課題やクイズの管理を効率化し、学習体験を向上させるブラウザ拡張機能

今すぐダウンロード GitHub

主な機能

🎨

カラーコーディング

課題の期限に応じてコースサイトのタブを色分け表示。一目で緊急度がわかります。

  • 期限1日以内
  • 期限5日以内
  • 期限14日以内
  • 期限14日超
🔔

通知バッジ

未確認の最新課題を通知バッジで表示。重要な課題を見逃しません。

📋

miniSakai

すべての課題とクイズを一覧表示。カスタムメモも追加でき、チェックボックスで進捗管理も可能。非表示・提出済みタブで効率的な課題管理を実現。

📅

カレンダー同期

Googleカレンダーとの連携機能により、課題の締切をカレンダーアプリで管理可能。

📝

メモ・資料管理

講義ごとのメモ機能と外部リンク管理。フォルダ機能で講義資料の閲覧・一括ダウンロードも可能。

🗓️

時間割表示

講義を時間割形式で表示。直感的なインターフェースで講義サイトへの移動も簡単。
